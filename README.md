Outsell Sharing
===============

This is a Drupal 7 module that handles Outsell contnet sharing

Getting Started
---------------

Clone the project and install dependencies to get ready for development

```bash
git clone git@github.com:teamoutsell/outsell_sharing.git
cd outsell_sharing
composer install
```

Testing
-------

```bash
composer test
```
